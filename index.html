playing the game

Set is a game about finding patterns. Your goal is to find sets of three cards. Each card has four attributes-- color, shade, fill, and number. A set is a group of cards which across each of those four attributes is either all the same (eg, all red, all diamond, all solid) or all different (eg, one red, one green, & one purple).
